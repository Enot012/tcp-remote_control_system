TCP Remote Control System
Асинхронная клиент-серверная система для удалённого управления компьютерами через TCP.
Возможности:

	Удалённое управление
	CMD - Выполнение команд на клиентах
	SIMPL - Пакетное выполнение команд из файла
	IMPORT/EXPORT - Двусторонняя передача файлов
	Отложенные команды

Автоматическое выполнение команд при подключении клиента
Поддержка целей: конкретный пользователь, группа, все клиенты
Отслеживание статуса выполнения
Плейсхолдер {User} для подстановки имени клиента

Система групп:

	Создание групп пользователей
	Массовая отправка команд группе
	Управление участниками

Мониторинг:

	Отслеживание активных команд
	Таймауты и предупреждения
	История выполнения
	Логирование всех действий

Ключевые особенности:

	Асинхронная архитектура (asyncio) - одновременная работа с множеством клиентов
	Флаг all - массовая отправка команд всем подключённым
	Автореконнект - клиент автоматически переподключается при обрыве
	Сохранение результатов - история выполнения команд
	Переименование пользователей (alias) для удобной работы
	Защита от сбоев - изоляция ошибок, graceful degradation

Файлы
server_v3_SIMPL_REALLY_FIXED.py
Сервер управления. Принимает подключения клиентов, отправляет команды, получает результаты, управляет отложенными командами и группами.
Основные команды:
CMD <client|all> <команда>        # Выполнить команду
simpl <client|all>                # Команды из code.txt
import <client|all> <путь> <путь> # Отправить файл
export <client> <путь> [dir]      # Получить файл
kick <client|all>                 # Отключить клиента
rename <user> <alias>             # Переименовать
chart_new                         # Создать отложенную команду
group_new <name>                  # Создать группу
list                              # Список пользователей
help                              # Все команды
client_v3_SIMPL_ALL_FIXED.py
Клиент. Подключается к серверу, выполняет команды, отправляет результаты и файлы.
Особенности:

Автоматическое переподключение при обрыве связи
Чанкирование больших выводов для надёжной передачи
Обработка KICK команды с корректным завершением
Кодировка cp866 для корректной работы в Windows



Python 3.7+
asyncio - асинхронное программирование
socket - сетевое взаимодействие
json - хранение данных
